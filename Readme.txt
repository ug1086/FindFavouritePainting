How to run the application:

Extract the zip file and open the 'index.html' in a browser.

About the project:

The page has two sections. In the first section, User can fill in their name and age. Name and Age are required fields before a user can click on 'Submit'.The user's name and age gets stored to the local storage and is prepopulated on revisiting the page.

The second section of the page is to find your favourite painting. In order to search user will have to select the Country, then the Painter and finally the Painting. Options for the selects are populated from JSON objects, hard coded in Script1.js.

Finally the user's selected painting is displayed at the bottom.
